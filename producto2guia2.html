Lista de Definiciones

HTML
Lenguaje de marcado utilizado para estructurar páginas web.
CSS
Lenguaje de estilo utilizado para definir la apariencia de una página web.
JavaScript
Lenguaje de programación utilizado para crear interactividad en las páginas web.
Visual Studio Code
Editor de código fuente que soporta múltiples lenguajes de programación y tiene muchas extensiones útiles.
